# Support

InferenceMeter is a community-supported open-source project maintained on a best-effort basis.

## Get help

- Search the [issue tracker](https://github.com/MonomythDevelopment/inference-meter/issues) for an existing report.
- Use the bug report form for reproducible failures.
- Use the feature request form for proposed behavior.
- Use [SECURITY.md](SECURITY.md) for suspected vulnerabilities.

Please include the InferenceMeter version, macOS version, Claude Code or Codex version, expected behavior, and actual behavior. Provider formats change frequently, so CLI version information matters.

Never post credentials, Keychain exports, `auth.json`, authorization headers, prompts, or unsanitized rollout files. Maintainers will not ask for your access token or refresh token.

Questions about Claude Code accounts, Codex accounts, billing, or provider limits must go to the relevant provider. InferenceMeter is not affiliated with Anthropic or OpenAI and cannot change account limits.
